🏨 Booking Widget

Widget embebible para reservas de hotel - Ejemplo de integración

Widget de Reservas

Este es un ejemplo de cómo integrar el widget de reservas en tu sitio web. El widget se conecta automáticamente con el backend local en desarrollo.

📋 Cómo integrar

Para integrar el widget en tu sitio web, sigue estos pasos:

1. Incluir el script

<script src="https://tu-dominio.com/dist/booking-widget.js" data-tenant="hotel-luna" data-language="es" data-primary="#0EA5E9"></script>

2. Crear el contenedor

<div id="booking-widget"></div>

3. Inicializar el widget

<script> BookingWidget.mountWidget('#booking-widget', { // Opciones adicionales (opcional) }) </script>

⚙️ Configuración técnica

Parámetros disponibles

  • data-tenant: Identificador del hotel/tenant
  • data-language: Idioma del widget (es, en)
  • data-primary: Color primario en formato hex

API Endpoints utilizados

  • GET /api/v1/tenants/{tenant}/config - Configuración del tenant
  • GET /api/v1/availability - Búsqueda de disponibilidad
  • POST /api/v1/bookings/confirm - Confirmación de reserva

Navegadores soportados

  • Chrome 80+
  • Firefox 75+
  • Safari 13+
  • Edge 80+
